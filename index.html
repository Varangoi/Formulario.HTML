A) Na tag input com atributo type="text", para que serve o atributo "placeholder"?

R: O placeholderatributo especifica uma dica curta que descreve o valor esperado de um campo de entrada (por exemplo, um valor de amostra ou uma breve descrição do formato esperado).


B) Na tag input com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

R: Define uma caixa de seleção. As caixas de seleção permitem que um usuário selecione ZERO ou MAIS opções de um número limitado de opções.


C) Na tag input para que serve o atributo "value"?

R:O atributo especifica o valor inicial para um campo de entrada.


D) Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

R: input type="radio"


E) Para que serve a tag label? Para que serve o atributo "for" nessa tag?

R: tag define um rótulo para muitos elementos de formulário.O útil para usuários de leitores de tela, porque o leitor de tela lerá em voz alta o rótulo quando o usuário focar no elemento de entrada.


F) Na tag option para que serve o atributo "selected"?

R: Representa um controle que apresenta um menu de opções. As opções dentro do menu são servidas pelo elemento option, que podem agrupados por elementos . As opções podem ser pré-selecionadas para o usuário. optgroup


G) Nas tags de formulário para que serve o atributo "required"?

R: Esse é um atributo booleano usado para indicar que um determinando campo de formulário é obrigatório para o envio do mesmo.


H) Na tag form para que serve o atributo "action"?

R: O atributo action define o local (através de uma URL) ao qual serão enviados todos os dados recolhidos do formulário.


I) Na tag form para que serve o atributo "method"?

R: O atributo method define qual o método HTTP para enviar os dados (ele pode ser "GET" ou "POST").


J) Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form?

R: Atributo POST.


K) Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos?

R: O valor do atributo deve ser um id de um elemento form.
